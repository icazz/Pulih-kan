# Pulih.kan

Platform manajemen bantuan pemulihan berbasis web yang dirancang untuk mempercepat proses rehabilitasi bagi warga terdampak bencana (seperti banjir atau gempa). Platform ini menghubungkan warga yang membutuhkan bantuan perbaikan dengan tim admin/petugas secara transparan dan efisien.
